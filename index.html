GOS
Geometric Optic Simulator
Start
GOS (Geometric Optics Simulator) provides a framework to perform basic optical simulations based on the laws os what is known as "geometric optics". GOS is fully developed using Javascript and runs in your browser, it doesn't require downloading any executable or install anything.

List of features currently supported:
  • Intuitive user interface.
  • Save/Load your simulation results.
  • Realtime ray tracing.

  • Light sources supported:
    • Puntual light
    • Beam (colimated beam)

  • Elements supported:
    • Flat Mirror
    • Spherical Mirror
    • Converging Thin lense
    • Diverging Thin lense
    • Blocker
